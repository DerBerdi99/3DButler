3DButler - ERP Software

This project is hosted by me -- Hannes Berthold -- if this does matter to anyone.

Goals: 
Creates a frontend for a user in the customer role and for admins. (if you host the app, you will create your own admin account first)
Serves a webshop for the customer with filtering options, shopping options, cart and payment settings. (Selling Shop Products)
Serves a custom order service for prototyping, smaller production charges. (Selling Custom Order Products)
The administrator utilizes typical ERP - like functionalities for production, review of custom orders and file management.

For more information about the project, navigate the website folder. You'll find:
-The database (SQLite3 is used)
-css, icons, images and javascript. product_images for showcase
-admin and customer views (html templates)
-manager files for database-operations
-environment file, values must be changed
-DB-Commerce.py is a helper file for table drops / creates on the fly- can be deleted if not needed
